# Cloud-Based MSSP Infrastructure on AWS

## Project Overview
Led a 5-member team to design and deploy a secure cloud infrastructure 
simulating an MSSP environment. Conducted penetration testing, deployed 
Wazuh SIEM, and enforced NIST CSF + HIPAA compliance.

## Tools & Technologies
- AWS EC2, IAM, CloudTrail, S3
- Wazuh SIEM
- Nessus / Tenable for vulnerability assessment
- Metasploit, Kali Linux for penetration testing

## Key Outcomes
- Identified and remediated 12 critical misconfigurations
- Automated compliance checks against NIST CSF and HIPAA
- Centralized log aggregation and threat detection pipeline
